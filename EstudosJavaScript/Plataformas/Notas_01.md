# **O que são Judges?**

### Judges são plataformas online que permitem praticar programação ao resolver problemas e receber feedback imediato.

Ao desenvolver e submeter seu código, um “juiz” automático avalia se a solução está correta com base em testes previamente definidos. Essas dinâmicas são ideais para aprimorar suas habilidades de forma prática e contínua. 

Alguns exemplos de plataformas que oferecem dinâmicas assim são:

- BeeCrowd;
- Neps;
- LeetCode;
- HackerRank.

Essas plataformas oferecem uma ampla gama de desafios, desde os mais básicos até os mais avançados, permitindo que você evolua progressivamente.

> 10. CodinGame
- CodinGame é uma plataforma que oferece desafios de programação em um formato de jogo. Os usuários podem escolher entre uma variedade de linguagens de programação e resolver problemas para avançar no jogo. É uma maneira divertida e interativa de praticar suas habilidades de codificação.
