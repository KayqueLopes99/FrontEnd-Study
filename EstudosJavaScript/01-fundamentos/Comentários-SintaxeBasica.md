# Como Comentar:
- Mesma função da linguagem C.

```javascript
// Isto é um comentário
let x = 5; // Declare x, dê a ele o valor de 5
```

```javascript
/* 
Isto é um comentário
de várias linhas
*/
```

- Case sensitive (sensível a maiúsculas/minúsculas) significa que o JavaScript diferencia letras maiúsculas de minúsculas.

- Quando você declara uma variável fora de qualquer função, ela é chamada de variável global, porque está disponível para qualquer outro código no documento atual. 

- JavaScript é uma linguagem dinamicamente tipada. Isso significa que você não precisa especificar o tipo de dado de uma variável quando declará-la, e tipos de dados são convertidos automaticamente conforme a necessidade durante a execução do script. 

> A tabela a seguir lista os caracteres especiais que podem ser usados em strings no JavaScript.
| Caractere Especial | Descrição                | Exemplo de Uso         |
|-------------------|--------------------------|------------------------|
| `\n`              | Nova linha               | `"Olá\nMundo"`         |
| `\t`              | Tabulação                | `"Nome:\tJoão"`        |
| `\\`             | Barra invertida          | `"C:\\Users\\João"`
| `\"`             | Aspas duplas             | `"Ele disse: \"Oi!\""` |


### Compilar:

- node nomeDoArquivo.js para executar um arquivo JavaScript usando o Node.js no terminal.

``` js
npm i prompt-sync
npm init -y      
node program.js
```
